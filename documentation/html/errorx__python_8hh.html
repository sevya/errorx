py::list errorx::correct_sequences ( py::list &  sequence_list,
py::list &  germline_sequence_list,
py::list &  phred_score_list,
py::str &  errorx_base 
)

Runs the ErrorX prediction method on a set of sequences and returns the corrected sequences in a Python list. Note: the sequences in sequence_list, germline_sequence_list, and phred_score_list are matched up element-wise, and must be the same length

Ex: sequence_list = ['ACTG','GTCA'] germline_sequence_list = ['ATTG','CTCA'] phred_score_list = ['GGFF','FFFG']

Parameters
sequence_listList of nucleotide sequences to be corrected
germline_sequence_listList of germline sequences corresponding to each sequence to be corrected
phred_score_listList of PHRED scores as strings to be corrected
errorx_baseThe root directory where the errorx code is located. This is set automatically by the Python code
Exceptions
invalid_argumentif lists are not the same length
Returns
Python list of corrected nucleotide sequences, corresponding element-wise to the input sequence list

Runs the ErrorX prediction method on a single sequence and returns the probability that each base is an error.

Ex: sequence = 'ACTG' germline_sequence = 'ATTG' phred_score = 'GGFF'

Parameters

Exceptions

Returns
Python list of tuples showing the probability of error for each base, each tuple representing (position, probability)
